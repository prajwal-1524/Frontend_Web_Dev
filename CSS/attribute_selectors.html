1. [attribute] Selector
Selects elements with the specified attribute, regardless of value.
[data-required] { border-left: 3px solid #e53e3e; }
This paragraph has the data-required attribute
2. [attribute="value"] Selector
Selects elements with the exact specified attribute value.
[type="submit"] { background-color: #2ecc71; color: white; }
3. [attribute*="value"] Selector
Selects elements whose attribute value contains the specified word in a space-separated list.
[class*="highlight"] { background-color: #f1c40f; }
This paragraph has the highlight class
This paragraph doesn't have highlight
This won't be selected (no exact match)
4. [attribute^="value"] Selector
Selects elements whose attribute value is exactly "value" or begins with "value-".
[lang^="en"] { border-bottom: 2px dotted #9b59b6; }
English content
American English content
British English content
French content (not selected)
